Tipos Primitivos

  • String : " isso é uma String"
  • Number : 1,2,3,4 ...
  • boolean : true , false
  • undefined : indefinido (não encontrado)
  • null : sem valor

O typeof pode ser usado para verificar qual o tipo do dado criado exemplo:
teste / ele retornará que o tipo de dado é String
enquanto se for atribuido da seguinte maneira
teste = 1 / o typeof apresentará ele como number
e assim por diante com os outros tipos primitivos
